# Fast-Food-delivery-database
Fast Food delivery database build using MySQL

Build a dummy database for swiggy with multiple tables and try to find an answer for the following question:

  1.Write a SQL query to find the number of Swiggy users
  
  2.Write a SQL query to find details of Swiggy delivery Boy

  3.Write a SQL query to find  the list of Swiggy users who made more than 10 orders in a particular month

  4.Write a SQL query to find top 10 Swiggy delivery Boy on basis of customer rating and time to deliver the item

  5.Write a SQL query to find the list of Swiggy users who order food from the same restaurants more than 3 times in a week
